1. Normal Distribution – Acres Burned

import numpy as np
from scipy.stats import norm

mu = 4300   # mean
sigma = 750 # standard deviation
x1 = 2500   # lower bound
x2 = 4200   # upper bound

# cumulative probabilities
p1 = norm.cdf(x2, mu, sigma)
p2 = norm.cdf(x1, mu, sigma)

probability = p1 - p2
print(f"Probability: {probability:.4f}")

Explanation: - This code calculates the probability that acres burned fall between 2500 and 4200. - norm.cdf gives the cumulative probability up to a point. - Subtracting gives the probability between two bounds. - Result ≈ 0.4358 (43.58%).

Extra Variation: You could use norm.sf() (survival function) for upper tail probabilities.

2. Discrete Distribution – Find k, Mean, Variance

import numpy as np

x_values = np.array([-2, -1, 0, 1, 2])
p_values = np.array([1/8, 2/8, None, 2/8, 1/8])

# find missing k
k = 1 - sum([p for p in p_values if p is not None])
p_values[2] = k

# mean and variance
mean = np.sum(x_values * p_values)
variance = np.sum((x_values - mean) ** 2 * p_values)

# probability range
p_range = p_values[1] + p_values[2] + p_values[3] + p_values[4]

print(f"k: {k:.3f}")
print(f"Mean: {mean:.3f}")
print(f"Variance: {variance:.3f}")
print(f"P(-1 < x ≤ 2): {p_range:.3f}")

Explanation: - Ensures probabilities sum to 1 by solving for k. - Mean = expected value of X. - Variance = spread of distribution. - Range probability adds up probabilities for -1 < x ≤ 2. - Results: k=0.25, Mean=2, Variance=1.5, Probability=0.625.

Extra Variation: Use np.dot(x_values, p_values) for mean directly.

3. Poisson Distribution – Failures

from scipy.stats import poisson

lambda_1h = 0.02  # average failures per hour

# (a) No failures in 8 hours
p_no_fail_8h = poisson.pmf(0, lambda_1h * 8)

# (b) At least one failure in 24 hours
p_at_least_1_24h = 1 - poisson.pmf(0, lambda_1h * 24)

print(f"No failure (8h): {p_no_fail_8h:.4f}")
print(f"At least one failure (24h): {p_at_least_1_24h:.4f}")

Explanation: - Poisson models rare events. - pmf(0) gives probability of zero events. - Subtract from 1 to get probability of at least one event.

Extra Variation: Use poisson.cdf(k, λ) for cumulative probabilities.

4. Binomial Distribution – Coin Toss Fit

import numpy as np
from scipy.stats import binom

x = np.arange(0, 9)
observed_freq = np.array([2, 6, 24, 63, 64, 50, 36, 10, 1])
total_throws = sum(observed_freq)

n = 8
p = 0.5

pmf = binom.pmf(x, n, p)
expected_freq = pmf * total_throws

print("x\tObserved\tExpected")
for x_, o, e in zip(x, observed_freq, expected_freq):
    print(f"{x_}\t{o}\t{e:.2f}")

Explanation: - Fits binomial distribution to coin toss data. - binom.pmf gives probability of each outcome. - Expected frequencies = probability × total trials. - Compares observed vs expected.

Extra Variation: Use binom.rvs(n, p, size=1000) to simulate coin tosses.

5. Hypothesis Testing – Lung Capacity

from scipy.stats import ttest_ind

male_lungcap = df[df['Gender'] == 'male']['LungCap']
female_lungcap = df[df['Gender'] == 'female']['LungCap']

t_stat, p_value = ttest_ind(male_lungcap, female_lungcap, equal_var=False)

print(f"T-statistic: {t_stat:.4f}")
print(f"P-value: {p_value:.4f}")

if p_value < 0.05:
    print("Reject H₀: Significant difference.")
else:
    print("Fail to reject H₀: No significant difference.")

Explanation: - Performs two-sample t-test. - Null hypothesis: no difference in means. - If p-value < 0.05, reject null → significant difference. - Result: males and females differ significantly in lung capacity.

Extra Variation: Use ttest_rel for paired samples, or anova for multiple groups.
