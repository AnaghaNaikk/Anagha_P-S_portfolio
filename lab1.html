Overview

This lab demonstrates data inspection, preprocessing, visualization, skewness analysis, measures of central tendency and dispersion, and outlier detection using histograms and boxplots in Python.

  • Goal: Identify distribution patterns, detect outliers, measure variability, and summarize data.
  • Techniques: pandas, numpy, matplotlib, seaborn, scipy.

Libraries and starter checks

Core libraries

  • Python 3.x
  • pandas — data loading, cleaning, manipulation
  • numpy — numerical operations and arrays
  • matplotlib — plotting
  • seaborn — statistical plots
  • scipy — skewness and stats
