Histograms

Histograms visualize frequency distributions of continuous variables. Below are two examples: used cars selling price and diamond prices.

Used cars — Selling Price

Histogram of Selling Price of Used Cars (bins=20, KDE enabled). Counts are annotated above bars in the image.

# Reproduce:
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

df = pd.read_csv("used_cars.csv")
plt.figure(figsize=(8,5))
sns.histplot(df["Selling_Price"], bins=20, kde=True, color="#5271FF")
plt.title("Histogram of Selling Price of Used Cars")
plt.xlabel("Selling Price (in lakhs)")
plt.ylabel("Frequency")
plt.tight_layout()
plt.show()
Typical result: highest frequency appears in the lower price range (e.g., ~0–5 lakhs) depending on your dataset.

Diamonds — Price distribution

Distribution of Diamond Prices (pink bars, black edges, bins=30). Selected bar counts are annotated in the image.

# Reproduce:
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("diamonds.csv")
plt.figure(figsize=(9,5))
plt.hist(df["price"], bins=30, edgecolor="black", color="pink")
plt.xlabel("Price")
plt.ylabel("Frequency")
plt.title("Distribution of Diamond Prices")
plt.tight_layout()
plt.show()
Typical result: right-skewed distribution with most values at the lower end, tailing off to higher prices.

Boxplots — outlier detection and group comparison

Marks — Outlier Detection

Boxplot of Marks — Outlier Detection. Outliers and summary values (mean/median) are annotated in the image.

# Reproduce:
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

df = pd.read_csv("marks.csv")
plt.figure(figsize=(7,5))
sns.boxplot(y=df["Marks"], color="#8BD3E6")
plt.title("Boxplot of Marks — Outlier Detection")
plt.ylabel("Marks")
plt.tight_layout()
plt.show()

Carat vs Clarity — Grouped Boxplot

Distribution of Carat Weight by Clarity Level (grouped boxplots). Medians are annotated in the image.

# Reproduce:
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

df = pd.read_csv("diamonds.csv")
plt.figure(figsize=(10,6))
sns.boxplot(x="clarity", y="carat", data=df, palette="Pastel1")
plt.title("Distribution of Carat Weight by Clarity Level")
plt.xlabel("Clarity")
plt.ylabel("Carat Weight")
plt.tight_layout()
plt.show()
Compare medians and spreads across clarity categories to see differences in carat distributions.

Central tendency and dispersion

Compute common summary statistics for numeric columns (example uses a 'Marks' column).

# Central tendency
mean = df['Marks'].mean()
median = df['Marks'].median()
mode = df['Marks'].mode()

# Dispersion
variance = df['Marks'].var()
std_dev = df['Marks'].std()
data_range = df['Marks'].max() - df['Marks'].min()
