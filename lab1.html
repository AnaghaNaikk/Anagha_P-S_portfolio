Chapter 1 · Description of Data (Updated)

Introduction

Descriptive statistics organizes, summarizes and presents data so patterns and features are easy to see. This chapter adds concise topic cards and clearer visual examples for quick study.

Types of Data (brief)

  • Qualitative (Categorical): Nominal (no order), Ordinal (ordered)
  • Quantitative: Discrete (counts), Continuous (measurements)
  • Binary: Two possible values (yes/no)
  • Time series: Observations ordered in time
  • Cross-sectional: Observations at one point in time

Data Collection & Cleaning (brief)

  • Sources: surveys, experiments, administrative records, sensors
  • Sampling: random, stratified, cluster, convenience
  • Cleaning: handle missing values, remove duplicates, correct errors
  • Validation: check ranges, consistency, and outliers

Measures of Central Tendency (brief)

  • Mean: arithmetic average
  • Median: middle value (robust to outliers)
  • Mode: most frequent value
  • Grouped formulas shown in original notes; use median for skewed data.

Measures of Dispersion (brief)

  • Range: max − min
  • Variance (sample): \(s^2 = \dfrac{\sum (x - \bar{x})^2}{n-1}\)
  • Standard deviation: \(s = \sqrt{s^2}\)
  • IQR: Q3 − Q1 (used with box plot)

Shape of Distribution

  • Skewness: asymmetry (positive = right tail)
  • Kurtosis: tail heaviness (leptokurtic/platykurtic)
  • Normality: bell-shaped; many tests and plots check this

Correlation & Association

Measure linear association with Pearson's r; visualize with a scatter plot.

Index Numbers & Time Series (brief)

  • Index: relative change from a base period
  • Time series: trend, seasonality, cyclic patterns

Other Brief Topics (quick reference)

Percentiles & Quartiles

Divide ordered data into 100 (percentiles) or 4 (quartiles) equal parts.

Stem-and-Leaf

Compact display preserving original data values and distribution shape.

Relative Frequency

Frequency divided by total count; useful for comparisons.

Outliers

Values far from others; detect with IQR or z-scores.

Standard Error

Estimate of sampling variability of a statistic (e.g., mean).

Confidence Interval

Range likely to contain a population parameter with given confidence.

What I Learned in This Chapter

Data classification, central tendency, dispersion, distribution shape, correlation, and multiple graphical methods. Use the histogram for distribution, ogive for cumulative frequencies, box plot for spread and outliers, scatter for association, and pie for categorical shares.

Tip: For real datasets, compute numeric summaries (mean, median, SD, IQR) before interpreting graphs.
