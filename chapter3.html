Chapter 3 – Random Variables & Probability Distributions

Introduction to Random Variables

A random variable is a function that assigns a numerical value to each outcome of a random experiment. Random variables convert qualitative outcomes into quantitative form so probability laws can be applied.

Click to view example

If X denotes the number of heads obtained when tossing a coin three times, possible values of X are {0,1,2,3}.

Types of Random Variables

Random variables are classified into discrete and continuous based on the nature of their possible values.

Click to view explanation & examples

Discrete Random Variable: Takes countable values.
Examples: Number of defective chips, number of heads in coin toss.

Continuous Random Variable: Takes values over an interval.
Examples: Time to complete a task, height, weight.

Probability Distribution (Discrete)

A probability distribution describes how probabilities are assigned to the values of a discrete random variable. The sum of all probabilities must be 1.

Cumulative Distribution Function (CDF)

CDF gives the probability that the random variable takes a value less than or equal to x. It is non-decreasing and ranges from 0 to 1.

Formula: F(x) = P(X ≤ x)

Example: Graph of cumulative distribution (step graph style)

For example, F(1)=0.70 means there is a 70% chance that the sample has at most 1 defective chip. The CDF is non-decreasing and reaches 1 at X=4.

Mean of a Random Variable

The mean (expected value) represents the long-run average value of the random variable.

Click to view formulas & example

Discrete: E(X) = Σ x·P(x)
Continuous: E(X) = ∫ x f(x) dx

Example: Expected number of successes in a binomial experiment.

Variance and Standard Deviation

Variance measures the spread or dispersion of a random variable around its mean. Standard deviation is the square root of variance.

Click to view formulas

Variance: Var(X) = E(X²) − [E(X)]²
Standard Deviation: σ = √Var(X)

Poisson Distribution

Poisson distribution models the number of events in a fixed interval of time or space. Mean and variance = λ.

Formula: P(X = x) = (e⁻ˡ λˣ) / x!

Example: Number of failures of a testing instrument in 8 hours.
